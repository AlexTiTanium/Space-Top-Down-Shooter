# Space Top Down Shooter (STDS)

One of the Infinite attempts to write multiplayer top down shooter

Installation:
-----------------------------------------------------------------------

1. Install webpack

```
npm install webpack -g
```

2. Install dependencies

````
npm install
````

3. Profit!

Webpack usage:
-----------------------------------------------------------------------

Create build:

````
webpack
````

Create small build:

````
webpack -p
````

Compile my code in real time:

````
webpack --watch
````
